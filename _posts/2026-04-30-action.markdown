<h3>Introduction</h3>

A core challenge in video understanding is whether self-supervised models learn features that are useful for temporal reasoning. In this work, we investigated this question directly for V-JEPA2, Meta's state-of-the-art joint-embedding video model.

---

<h3>Methodology</h3>

Rather than relying on a single evaluation protocol, we benchmarked V-JEPA2 features using two complementary approaches: a trained probe via ActionFormer, a leading action localization architecture, and a training-free method based on clustering with Hungarian matching. Running both in parallel allowed me to analyze how well the two regimes correlate, essentially asking whether a model that looks strong under supervision also holds up when no labels are available. This dual-probe methodology offers a more robust signal of true feature quality than either approach alone.
